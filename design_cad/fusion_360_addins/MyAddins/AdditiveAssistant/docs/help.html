How to get started

Now you have the Additive Assistant (FFF) tool installed on your machine, now all you need to do is enable it in Fusion 360.

  • In the Design workspace, navigate to the "Tools" tab.
  • In the "Add-ins" group, press the "Scripts and Add-ins" icon.
  • Select the Add-ins tab in the window that opened.
  • Select the "AdditiveAssistant" item in the list, then press run.
  • Now return to the "Solid" tab in the "Design" workspace.
  • The new Additive Assistant icon should be visible in the "Inspect" group.

    • How to use

    Now that you have the Additive Assistant tool installed and running, you are ready to start taking advantage of it.

  • Firstly, design your part that you intend to 3D print using the FFF process.
  • Press the Additive Assistant icon to bring up the Setup window.
  • Follow the instructions in the window and select your BRep body, your material, input the build area of your printer and then press Inspect.
  • Get an immediate idea of how suitable your part is using the result column of the table.
  • For more information about each aspect of analysis, select the "i" icon.
  • When you have finished inspecting the feedback, click the Close button.
