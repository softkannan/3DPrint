Object Resizer Usage Instructions

Object Resizer add-in can be used to resize an object using dimensional values


Steps:

1. Press the Object Resizer button in the "Sketch" panel.

2. Select the body.

3. Enter the expected values in appropriate fields.

a) Write value along the x axis and other coordinates will be calculated automatically.

b) Write value along the y axis and other coordinates will be calculated automatically.

c) Write value along the z axis and other coordinates will be calculated automatically.

4. Select point of the body.

5. Press "OK" button.

Check if the body dimensions has changed.
