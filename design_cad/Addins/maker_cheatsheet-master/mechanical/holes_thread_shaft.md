### Common Cutting Speed

| Material | Cutting speed (m/min)
|:--:|:--:|
| Softwood | 65 |
| Hardwood | 50 |
| Steel (tough) | 15 |
| Steel (mild) | 25 |
| Aluminum | 75 |
| Brass | 40 |
| Stainless Steel | 25 |
| Copper/Bronzes | 30 |


### Formula
n = 1000 * Vc / ( pi * d )

Where:\
n = Rotational Speed of the Main Spindle (rotation/min)\
Vc = Cutting speed, depends on material, see table bellow (m/min)\
d = drill diameter (mm)\
